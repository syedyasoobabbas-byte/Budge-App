# Budget App

A Python-based budget management application that helps users track income and expenses across multiple budget categories. Built using Object-Oriented Programming (OOP), the application supports deposits, withdrawals, fund transfers, balance tracking, and a text-based spending chart.

## Features

* Create multiple budget categories
* Deposit and withdraw funds
* Transfer money between categories
* Automatic balance calculation
* Check available funds before transactions
* Display a formatted transaction ledger
* Generate a percentage spending chart by category

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Math Module

## Concepts Practiced

* Classes and Objects
* Encapsulation
* Methods
* Lists and Dictionaries
* String Formatting
* Data Processing

## How to Run

1. Clone the repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file.

```bash
python budget.py
```
